# 🐞 Log4Scan 
🔧 Log4Shell 简单的主动和被动扫描脚本

## Log4scan
针对header头和fuzz参数的主动批量扫描，用于大批量黑盒检测
> 先配置ceye信息
```
self.host = ""
self.token = ""
```

burp suite 被动扫描插件，对每个参数、header头、Cookie做扫描
> 先配置ceye信息
```
self.host = ""
self.token = ""
```
